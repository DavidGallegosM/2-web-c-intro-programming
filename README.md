Este proyecto es una página web sencilla que creé como parte de mi entrega final. Aquí puse en práctica todo lo que aprendí sobre HTML, el uso de etiquetas semánticas, enlaces, imágenes, formularios y la estructura básica de un sitio web.
Además, utilicé Git y GitHub para guardar el historial de cambios y publiqué la página usando GitHub Pages para que cualquiera pueda verla en línea.

* ¿Qué contiene este proyecto?
El proyecto está formado por:
index.html — Es la página principal del sitio.
proyecto.html — Página donde explico de qué trata el proyecto.
contacto.html — Incluye un formulario simple de contacto.
css/style.css — Archivo con estilos básicos para mejorar la apariencia del sitio.
Carpeta “images” — Contiene las imágenes que usa la página.

* ¿Qué se puede ver en la página?
Dentro del sitio encontrarás:
Títulos y subtítulos para organizar la información.
Párrafos que explican quién soy y de qué trata el proyecto.
Enlaces para navegar entre las páginas.
Una imagen ilustrativa.
Un formulario sencillo con campos como nombre, correo y mensaje.
Un menú de navegación básico.
Todo está hecho usando únicamente HTML y CSS.
